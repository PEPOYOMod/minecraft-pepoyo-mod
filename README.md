[h1=Description]

This is a Minecraft mod featuring PEPOYO's (Japanese VOCALOID producer) characters, including 9 playable roles.

[h2=📕 Add content]

Added Yande,Yorupo,POYOROID,Waraninja,other creatures and spawn eggs.Some creatures can naturally generate.

Use hammer right click separately Waraninja and POYOROID can summon two bosses,Use ramen right click Yorupo can summon boss too.
